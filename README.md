# Shilo Claude Plugins

Plugin marketplace for Claude Code.

## Install Marketplace

```
/plugin marketplace add shilo/claude-plugins
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [`claude-loves-shelly`](https://github.com/shilo/claude-loves-shelly) | Run shell commands with `!` prefix or `/shelly` command |

Install a plugin:
```
/plugin install claude-loves-shelly@shilo
```
